======================
Learn K8S
======================

Introduction
============

This guide covers the fundamentals of Kubernetes (K8S).
What this Repo not is :
- A complete Kubernetes course
- A replacement for official documentations
- A deep dive into advanced Kubernetes topics

Topics
======

- Pods
- Services
- Deployments
- ConfigMaps and Secrets
- Ingress
- Volumes

Getting Started
===============

Prerequisites
-------------

- Docker installed
- kubectl CLI tools
- A Kubernetes cluster (local or cloud)

Installation
------------

Look into the different parts like 01_KIND for local cluster setup using kind,
or 02_MINIKUBE for minikube setup.

Follow the official `Kubernetes documentation <https://kubernetes.io/docs/setup/>`_ for setup instructions.
